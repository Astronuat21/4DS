Hallo! Welcome to the Github repository for the 4DS (nintendo dont sue)

4DS is a standalone project by ME aiming to make a dual screen console _kinda like the Nintendo DS's_ but with stronger hardware, better battery life and specs in general. The goal is to take the large library and amazing
portability, and refresh it with more beefy specs. 

This repository contains all the PCB files used in this project, including schematics and circuit boards. **Assuming everything works,** you should be able to reproduce the exact boards I used and reprint them for yourself. 
Also included are all the STL files used in 3D printing _(Which is ABS-GF filament, BTW)_
